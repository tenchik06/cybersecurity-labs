# Задание 5: Расшифровка Base64 сообщения

## Текст задания
При сканировании сетевого трафика был перехвачен подозрительный пакет данных. Внутри пакета обнаружена строка в кодировке Base64. Расшифруйте сообщение и найдите скрытый флаг.

**Формат флага:** FLAG{...}

## Файлы
- `encoded_file.txt` - закодированное сообщение

## Подсказки
- Просто декодируйте строку из Base64
- В расшифрованном тексте будет флаг
