# Задание 1: Подозрительный файл

## Контекст
Вы получили подозрительный файл от сотрудника. Внутри файла спрятано секретное сообщение. Найдите его и получите флаг.

## Исходные данные
Файл: `suspicious_file.txt`

## Задача
Найти скрытое сообщение и представить его в формате флага: `FLAG{...}`

## Подсказки
- Обратите внимание на форматирование текста
- Заглавные буквы могут содержать важную информацию
