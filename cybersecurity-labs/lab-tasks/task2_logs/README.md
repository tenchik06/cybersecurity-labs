# Задание 2: Анализ логов доступа

## Текст задания
В логах веб-сервера обнаружена подозрительная активность. Найдите IP-адрес злоумышленника и количество попыток доступа.

**Формат флага:** FLAG{IP_кол-во попыток}

## Файлы
- `access_logs.txt` - логи доступа к веб-серверу

## Подсказки
- Ищите IP, который обращается к защищенным разделам (/admin, /wp-admin)
- Посчитайте количество запросов с этого IP
