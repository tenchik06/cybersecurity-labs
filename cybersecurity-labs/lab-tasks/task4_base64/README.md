# Задание 4: След злоумышленника

## Текст задания
В логах системы обнаружена подозрительная строка в кодировке Base64. Расшифруйте её, чтобы узнать, что пытался скрыть злоумышленник.

**Формат флага:** FLAG{...}

## Файлы
- `base64_message.txt` - закодированное сообщение

## Подсказки
- Используйте любой декодер Base64
- Флаг уже находится внутри расшифрованного сообщения
