# cybersecurity-labs
# Лабораторная работа: Цифровая культура в профессиональной деятельности

## Разработка Task-based задач для проведения киберучений

### О работе

Данная лабораторная работа представляет собой набор практических заданий по информационной безопасности, разработанных для повышения осведомленности сотрудников о современных киберугрозах.

**Год:** 2025

### Содержание работы

Работа включает 5 практических заданий:

1. **Анализ подозрительного файла** — поиск скрытой информации в текстовом файле
2. **Анализ логов доступа** — выявление подозрительной активности веб-сервера
3. **Анализ вредоносного скрипта** — исследование Python-скрипта на Python
4. **Декодирование Base64** — расшифровка скрытого сообщения
5. **Расшифровка Base64 сообщения** — работа с закодированными данными

### Цель работы

Практическая подготовка студентов в области изучения проблем, связанных с различного рода уязвимостями в программном коде и методами социальной инженерии.

### Как использовать материалы

Каждое задание находится в отдельной папке `taskX_...` и содержит:
- Исходные файлы для анализа
- Краткое описание задания в README.md

Решения доступны в папке `solutions/answers.md`

### Требования

- Базовое понимание работы с текстовыми файлами
- Знание Python (для задания 3)
- Понимание принципов кодирования Base64
